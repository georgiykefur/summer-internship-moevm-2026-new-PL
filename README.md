# Визуализация алгоритма Флойда-Уоршелла

Десктопное приложение на java swing для визуализации алгоритма Флойда-Уоршелла нахождения кратчайших путей между всеми парами вершин во взвешенном ориентированном графе.


## Функциональные возможности

### Управление графом

- добавление вершин — двойной клик на холсте или кнопка "+вершина"
- добавление рёбер — через контекстное меню вершины (пкм → "добавить ребро от")
- удаление элементов — через контекстное меню (пкм на вершине или ребре)
- перемещение вершин — перетаскивание по холсту с зажатой левой кнопкой мыши
- масштабирование — кнопки + и − в левом нижнем углу или колесо мыши
- панорамирование — перетаскивание фона холста с зажатой левой кнопкой мыши

### Ввод/Вывод данных

- загрузка из файла — кнопка "загрузить"
- сохранение в файл — кнопка "сохранить"

### Визуализация алгоритма

- запуск — кнопка "выполнить" (полный проход алгоритма)
- пошаговый режим — кнопки "назад" / "вперёд"
- подсветка — вершины k (красный), i (синий), j (зелёный)
- обновление рёбер — улучшенные пути подсвечиваются зелёным, новые — пунктиром
- матрица расстояний — обновляется в реальном времени

### Взаимодействие с результатом

- просмотр пути — клик по ячейке финальной матрицы

### Компиляция и сборка

- javac *.java
- jar cfe FloydWarshall.jar Main *.class
- java -jar FloydWarshall.jar
